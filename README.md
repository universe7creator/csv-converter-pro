# CSV Converter Pro

Professional data transformation tool for developers. Convert between CSV, JSON, TSV, Markdown, HTML, and SQL formats instantly.

## Features

- **Multi-Format Support**: CSV ↔ JSON ↔ TSV ↔ Markdown ↔ HTML ↔ SQL
- **Smart Detection**: Auto-detects delimiters and data types
- **Real-time Stats**: Row count, column count, cell count, file size
- **Flexible Options**: Header rows, whitespace trimming, null handling
- **One-Click Actions**: Convert, copy, download
- **No Server Required**: Runs entirely in your browser

## Use Cases

- Convert API responses from JSON to CSV for Excel analysis
- Transform spreadsheet data into SQL INSERT statements
- Generate Markdown tables from CSV files
- Convert between different delimiter formats
- Prepare data for import/export operations

## Formats Supported

### Input
- CSV (Comma Separated Values)
- TSV (Tab Separated Values)
- JSON (Object or Array)

### Output
- JSON (Pretty printed or compact)
- CSV
- TSV
- Markdown Table
- HTML Table
- SQL INSERT statements

## Options

### CSV Options
- First row as header
- Trim whitespace
- Skip empty rows

### JSON Options
- Pretty print
- Use null for empty values
- Flatten nested objects

### Output Options
- Quote all strings
- Include row index
- Lowercase headers

## How to Use

1. Paste your data in the Input panel
2. Select input and output formats
3. Configure options as needed
4. Click "Convert"
5. Copy or download the result

## Technical Details

- Single HTML file - no installation required
- Pure JavaScript - no external dependencies
- Works offline after initial load
- Handles large files efficiently
- Responsive design for mobile and desktop
